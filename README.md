## Component list

* 1x Arduino pro micro atmega32u4 (native USB port not like Arduino nano)
* 3x 10k slide potentiometers
* 3x rotary encoders 
* 1x CD74HC4067 16-Channel Analog Digital Multiplexers
* 1x MC14490P hardware debouncer
* 1x 10nf capacitor
* 16X push buttons
